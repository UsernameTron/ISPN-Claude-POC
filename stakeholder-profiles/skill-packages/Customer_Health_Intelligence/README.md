# Customer Health Intelligence - Quick Start Guide

## What This Skill Does

Transforms reactive churn discovery (9-12 months too late) into proactive customer health monitoring with 90-180 day early warning system. Enables customer success teams to prevent churn, accelerate expansion, and optimize resource allocation across K-12 school district customer portfolio.

## Key Features

- **Customer Health Scoring**: Multi-factor analysis across support, engagement, usage, relationship, and sentiment
- **Churn Prediction**: 90-180 day advance warning with 72%+ accuracy for at-risk account identification
- **Retention Strategy Development**: Automated multi-phase retention roadmaps with executive escalation protocols
- **Expansion Opportunity Identification**: Proactive expansion pipeline with revenue potential and close probability
- **Automated QBR Preparation**: Customer-specific QBR materials in 20 minutes vs 3-4 hours manual prep

## Quick Start (5 Minutes)

### 1. Install Required MCPs
```bash
# Required MCPs (must have)
- Salesforce MCP (customer accounts, contracts, renewals)
- Zendesk MCP (support tickets, CSAT scores)
- Usage Data Integration (bandwidth, uptime, adoption)

# Optional but Recommended
- Email Integration (sentiment analysis, engagement tracking)
```

### 2. First Request
Try: `"Analyze customer health across all active accounts. Flag at-risk renewals in next 90 days."`

### 3. Common Requests
- `"Show me all Red accounts requiring executive intervention"`
- `"Deep-dive analysis for [Account Name] with retention strategy"`
- `"Identify expansion opportunities across Green accounts"`
- `"Prepare QBR materials for [Account Name]"`
- `"Quarterly customer health executive review for board"`
