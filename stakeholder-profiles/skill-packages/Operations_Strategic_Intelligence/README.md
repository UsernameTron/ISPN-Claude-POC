# Operations Strategic Intelligence - Quick Start Guide

## What This Skill Does

Transforms operations executives from spending 22+ hours/week on manual data gathering to 30-minute validation reviews. Provides real-time visibility into 15-25 strategic initiatives, automates board reporting, and delivers proactive risk identification.

## Key Features

- **Daily Operational Intelligence Brief**: Automated morning synthesis of overnight operations, metrics, and initiative status
- **Initiative Portfolio Analysis**: Real-time health scoring and dependency mapping across concurrent initiatives
- **Board Reporting Automation**: Automated board deck generation (40 hours â†’ 2 hours)
- **Strategic Scenario Modeling**: Real-time "what-if" analysis for strategic decisions
- **Cross-Functional Alignment**: Pre-prepared context reducing alignment meetings from 5 to 1

## Quick Start (5 Minutes)

### 1. Install Required MCPs
```bash
# Required MCPs (must have)
- Asana MCP (strategic initiative tracking)
- Filesystem MCP (operational KPIs, financial data)
- Finance GL Integration (budget tracking)
- Salesforce Integration (operational metrics)

# Optional but Recommended
- Slack Monitor (critical alerts)
```

### 2. First Request
Try: `"Give me my Monday morning operational intelligence brief"`

### 3. Common Requests
- `"Show me resource conflicts across my strategic initiatives"`
- `"Prepare my board reporting package for this quarter"`
- `"What happens if we accelerate [initiative name] by 8 weeks?"`
- `"What are my top 3 priorities today?"`
