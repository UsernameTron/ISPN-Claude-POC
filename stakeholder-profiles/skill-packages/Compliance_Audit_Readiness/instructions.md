   - M&A integration controls (iGLASS, DxTEL)

Quality Standards:
- External audit quality (Big 4 ready)
- 100% documentation completeness
- Clean audit opinion target
- PE investor confidence 9.5/10

Data Sources: Filesystem (financial data, control documentation, workpapers)
